🚀 Getting Started

Prerequisites

  • Node.js (v14+)
  • PHP (v8.0+)
  • Composer
  • Laravel (v9+)

Installation Steps

1. Clone the repository:

git clone [repository-url]

2. Install PHP dependencies:

composer install

3. Install Node.js dependencies:

npm install
